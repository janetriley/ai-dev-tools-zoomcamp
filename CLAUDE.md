## Python Development

**Python Version**: 3.12+

Use `python3.12` or `python3` for consistency.

### Virtual Environments

Create virtual environments within module directories as needed:
```bash
python3 -m venv .venv
source .venv/bin/activate  # On macOS/Linux
pip install -r requirements.txt
```

The `.gitignore` excludes common virtual environment directories (`.venv/`, `venv/`, etc.).

### Homework Workflow

When working on homework:

1. **Update course materials** at the start of each session:
   ```bash
   cd /Users/me/workspace/ai_dev_tools/main_course_repo && git pull && cd -
   ```

2. **Review official materials** in `/Users/me/workspace/ai_dev_tools/main_course_repo` for:
   - Homework instructions (markdown files)
   - Starter code (Python scripts, notebooks)
   - Sample data files
   - Module-specific requirements

3. **Copy README.md** from course repo to the matching `0X-*/` directory (do not copy sample code)

4. **Check evaluation criteria** at https://courses.datatalks.club/ai-dev-tools-2025/ before starting work

5. **Organize by module** - keep all work in the appropriate `0X-*/` directory

6. **DO NOT modify agent-generated code** without asking first - homework files often include code generated by agents and tools

7. **Reference FAQ** at https://docs.google.com/document/d/1uBSxORcxOewXMzMDHwADpVSiS0kBRXhTQ3qWd86CjlI/ for common questions
